# Hodgins.insure Landing Page

A responsive, high-converting landing page for Hodgins.insure, built with React and Tailwind CSS.

## Features

- **Hero Section** with bold headline, quote form, and primary CTA
- **Trust Badges** showing ratings and credentials
- **Savings Comparison** callout
- **How It Works** section with 3 simple steps
- **Benefits Section** highlighting key advantages
- **Testimonials** with customer review cards
- **Educational Section** about homeowners insurance
- **FAQ Section** with accordion functionality
- **Professional Footer** with branding, legal links, and consent disclaimer

## Getting Started

### Install Dependencies
```bash
npm install
```

### Run Development Server
```bash
npm run dev
```

The site will be available at `http://localhost:5173`

### Build for Production
```bash
npm run build
```

### Preview Production Build
```bash
npm run preview
```

## Tech Stack

- **React 18** - UI framework
- **Vite** - Build tool and dev server
- **Tailwind CSS** - Utility-first CSS framework
- **PostCSS** - CSS processing

## Project Structure

```
├── index.html          # Main HTML file
├── src/
│   ├── main.jsx       # React entry point
│   ├── App.jsx        # Main app component
│   └── index.css      # Tailwind CSS imports
├── package.json       # Dependencies and scripts
├── vite.config.js     # Vite configuration
├── tailwind.config.js # Tailwind configuration
└── postcss.config.js  # PostCSS configuration
```

## Customization

Edit `src/App.jsx` to customize content, colors, and layout. The design uses Tailwind's utility classes for easy customization.
